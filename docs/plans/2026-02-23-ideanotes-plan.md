# IdeaNotes 实现计划

**项目**: IdeaNotes - 跨平台手写笔记应用  
**日期**: 2026-02-23  
**版本**: 1.0.0

## 项目概述

一款支持手写输入与 OCR 识别的跨平台笔记应用，支持 iOS、Android、鸿蒙系统。

## 技术栈

- **框架**: Flutter 3.x
- **状态管理**: flutter_bloc
- **本地数据库**: sqflite
- **OCR 识别**: google_mlkit_text_recognition
